This folder contains the detailed merge information. 
The number of all classes in NABird dataset is 1011. 
Only 555 leaf classes are associated with image information. 
According to the hierarchy information, we further merge theses 555 leaf classes into 404 classes.



Classes.txt :   a list of all 1011 class names
hierarchy.txt:  the hierarchy of all classes. 
Mergetable.mat: information of all merge operations. 
Merge_log.txt:  a readable merge log showing all merge operations.
